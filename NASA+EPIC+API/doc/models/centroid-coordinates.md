
# Centroid Coordinates

The coordinates of the centroid are simply the average of the coordinates of the vertices.

## Structure

`CentroidCoordinates`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `lat` | `number` | Required | latitude |
| `lon` | `number` | Required | longitude |

## Example (as JSON)

```json
{
  "lat": -5.222168,
  "lon": 159.748535
}
```
